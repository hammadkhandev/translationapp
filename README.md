## Getting Started

To set up and run iTransaction locally:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/hammadkhandev/translationapp.git
   cd translationapp
   ```

2. **Install dependencies:**
   ```sh
   flutter pub get
   ```

3. **Run the app:**
   ```sh
   flutter run
   ```
